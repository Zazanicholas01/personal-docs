# Technology map

This documentation is organized as a cascade: macro category, topic area, then individual notes.

## Foundations

This branch covers the layers closest to the machine: hardware, operating systems, Linux, networking, and storage.

## Software engineering

This branch covers how software is written and structured: Git and GitHub, languages, frontend, backend, system design, and software practices.

## Platform infrastructure

This branch covers how software is deployed and operated: cloud, containers, Kubernetes, DevOps, observability, reliability, and security.

## Data and AI

This branch covers data systems and intelligent systems: databases, pipelines, AI concepts, and MLOps.

## Specialized domains

This branch covers domain-focused topics such as embedded systems, mobile, productivity tooling, VR/AR, and 3D design.
